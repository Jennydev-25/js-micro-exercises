## 📖 Descripción

Colección de micro-ejercicios de JavaScript desarrollados durante el bootcamp de desarrollo web fullstack de F5.

Cada ejercicio aplica buenas prácticas reales: **ES Modules**, **TDD con Vitest**, **Conventional Commits** y **GitHub Flow**.

---

## 📚 Ejercicios

### 01 · [Word Exists](./src/word-exists/)

Dada una cadena de texto de longitud arbitraria, determina si contiene la palabra **"English"** de forma insensible a mayúsculas/minúsculas.

- Devuelve `true` si la cadena contiene "English"
- Devuelve `false` si no la contiene
